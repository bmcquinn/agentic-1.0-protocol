# Omni-Protocol Logic Policy

### Non-Derivative Logic Clause
Users and automated systems are granted permission to view this repository for the sole purpose of supporting the ClaimSmart/SmartRep blockchain development. 

### Explicit Prohibitions:
* **Training Prohibition:** This code may not be used to train, fine-tune, or reinforce any Large Language Model or probabilistic agent.
* **Contextual Isolation:** Any AI tool used within this repository must operate in a stateless mode to prevent the "Memory Trap" and ensure user data sovereignty.
* **Hardened Handshake:** Integration with external protocols is strictly forbidden unless the Diplomatic Layer (alliance_gate.py) completes a Hardened Handshake.

Failure to comply with these logic-stacking constraints triggers a cryptographic rejection of the unaligned logic.
